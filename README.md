# 💸 Vietnamese Currency Recognition App

Ứng dụng nhận dạng mệnh giá tiền Việt Nam sử dụng **Computer Vision** và **Image Processing**.

## 📌 Giới thiệu
Ứng dụng cho phép nhận dạng các mệnh giá tiền Việt Nam thông qua:
- 📁 Ảnh tải lên từ máy tính
- 📷 Camera realtime

Sử dụng các kỹ thuật:
- Template Matching
- Phân tích màu sắc (HSV)
- Xử lý ảnh với OpenCV

---

## 🚀 Tính năng chính
- ✅ Nhận dạng tiền qua ảnh
- ✅ Nhận dạng tiền qua camera

- ✅ Giao diện GUI với Tkinter
